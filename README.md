# RPG Character Service

REST API сервис для управления RPG-персонажами.

---

## Описание

Приложение реализует HTTP-сервер с использованием Jetty и Jersey.
Позволяет создавать персонажей, получать их список и повышать уровень.

Данные хранятся в файловой базе данных H2 и сохраняются между перезапусками сервера.

---

## Функциональность

* создание персонажа
* получение списка всех персонажей
* получение персонажа по идентификатору
* повышение уровня персонажа

---

## Технологии

* Java
* Jetty
* Jersey (JAX-RS)
* JDBI
* H2 Database
* Maven

---

## Архитектура

Проект построен по слоистой архитектуре:

* `resource` — REST API
* `service` — бизнес-логика
* `repository` — работа с базой данных
* `model` — модели данных
* `filter` — логирование запросов
* `exception` — обработка ошибок
* `config` — конфигурация сервера

---

## Запуск

```bash
mvn clean compile exec:java
```

После запуска приложение доступно по адресу:

```
http://localhost:8080
```

---

## API

### Создание персонажа

```http
POST /characters
Content-Type: application/json
```

```json
{
  "name": "Arthas",
  "characterClass": "Warrior"
}
```

---

### Получение всех персонажей

```http
GET /characters
```

---

### Получение персонажа по id

```http
GET /characters/{id}
```

---

### Повышение уровня

```http
POST /characters/{id}/level-up
```

---

## База данных

Используется файловая база данных H2.

Файл базы данных создаётся автоматически при первом запуске приложения
и сохраняется в корне проекта.

Данные сохраняются между перезапусками и удаляются только вручную.
